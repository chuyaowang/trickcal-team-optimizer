# 用户指南

欢迎使用 **trickcal-team-optimizer** 宠物派遣计算器！本工具旨在帮助您找到最优的宠物派遣方案，以最大化您的派遣奖励。

## 选择界面

本程序提供三种交互方式：
1.  **网页界面 (推荐)**: 体验最现代，支持多语言切换和配置保存。
2.  **基础图形界面**: 使用 Tkinter 开发的轻量级桌面窗口。
3.  **命令行界面 (CLI)**: 适合高级用户或通过配置文件快速运行。

---

## 1. 网页界面 (Streamlit) 🌐

运行指令: `streamlit run src/ui/web_gui.py`

### 快速上手
1.  **语言切换**: 在左侧边栏顶部可切换**简体中文**或 **English**。
2.  **选择服务器**: 选择 CN, GL-CN, GL-EN 或 KR。
    - *注: GL-EN 使用英文的宠物和任务名称。*
3.  **选择任务文件**: 从下拉菜单中选择当前的任务列表。
4.  **选择您的宠物**:
    -   **我拥有的宠物**: 在双列网格中勾选仓库中的宠物（支持搜索）。
    -   **借用宠物数量**: 在三列网格中设置好友可借出的副本数。
5.  **设置任务上限 (P)**: 选择同时执行的任务数量（2-5）。

### 配置管理 💾
您可以保存配置以避免重复勾选：
-   **保存**: 点击侧边栏的 **“📥 下载当前配置”**。
-   **读取**: 在侧边栏上传 `.json` 配置文件即可恢复所有勾选和设置。

---

## 2. 基础桌面界面 (Tkinter) 🖥️

运行指令: `python -m src.ui.gui`

这是一个标准的桌面窗口版本。

### 使用说明
1.  **服务器与任务**: 在窗口顶部进行选择。
2.  **选择宠物**: 
    -   在“拥有宠物”列表中点击选择。
    -   在“农场宠物”表格中，**双击**宠物的“数量”列并输入副本数。
3.  **开始计算**: 点击按钮，结果将显示在下方。

---

## 查看结果

无论使用哪种界面，结果均包含：
-   **核心指标**: 显示总计层级奖励分和借用宠物总数。
-   **任务详情**: 每一个推荐的任务队伍及其原始得分、预计奖励等级（如：特阶、一阶、S 或 A）。

## 小贴士
-   计算器优先考虑跨越奖励阈值（如 37 分），而不是单纯堆砌原始分数。
-   在所有活跃任务中，总共最多只能借用 3 只宠物。
