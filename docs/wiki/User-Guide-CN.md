# 用户指南

欢迎使用 **ddl-PetDispatch** 宠物派遣计算器！本工具旨在帮助您找到最优的宠物派遣方案，以最大化您的派遣奖励。

## 快速上手

1.  **选择服务器**: 在 CN、GL 或 KR 中选择，确保数据与您的游戏版本一致。
2.  **选择任务文件**: 从下拉菜单中选择当前激活的任务列表。
3.  **选择您的宠物**:
    -   在**拥有宠物**列表中，勾选您仓库中现有的所有宠物。
    -   在**农场宠物**列表中，找到好友可借出的宠物。双击“数量”列，设置您可以借用的该宠物数量。
4.  **设置任务上限 (P)**: 选择您想同时执行的任务数量（通常在 2 到 5 之间）。

## 执行计算

点击 **“🚀 开始计算最优派遣方案”** 按钮。计算器将使用混合整数线性规划（MILP）分析成千上万种可能的组合，为您找到全局最优解。

## 查看结果

-   **总计层级奖励分**: 所有任务达成的奖励等级分值总和。
-   **任务详情**: 对于每一个选中的任务，工具将显示：
    -   推荐的派遣队伍。
    -   宠物是否为借用（标有“借”字）。
    -   任务原始得分及预计奖励等级（例如：特阶、一阶、S 或 A）。

## 小贴士
-   计算器优先考虑跨越奖励阈值（如 37 分），而不是单纯堆砌原始分数。
-   在所有任务中，总共最多只能借用 3 只宠物。
