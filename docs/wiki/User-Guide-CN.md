# 用户指南

欢迎使用 **ddl-PetDispatch** 宠物派遣计算器！本工具旨在帮助您找到最优的宠物派遣方案，以最大化您的派遣奖励。

## 选择界面

本程序提供三种交互方式：
1.  **网页界面 (推荐)**: 体验最现代，对 Linux 用户字体支持最完美。
2.  **基础图形界面**: 使用 Tkinter 开发的轻量级桌面窗口。
3.  **命令行界面 (CLI)**: 适合高级用户或自动化脚本运行。

---

## 1. 网页界面 (Streamlit) 🌐

运行指令: `streamlit run src/ui/web_gui.py`

### 快速上手
1.  **选择服务器**: 在左侧边栏选择 CN、GL 或 KR。
2.  **选择任务文件**: 从下拉菜单中选择当前激活的任务列表。
3.  **选择您的宠物**:
    -   **我拥有的宠物**: 勾选您仓库中现有的宠物（支持搜索）。
    -   **借用宠物数量**: 在列表中通过数字输入框设置好友可借出的宠物副本数。
4.  **设置任务上限 (P)**: 选择您想同时执行的任务数量（通常在 2 到 5 之间）。

### 配置管理 💾
您可以保存您的宠物配置，避免每次重复勾选：
-   **保存配置**: 点击左侧边栏的 **“📥 下载当前配置”** 下载 `.json` 文件。
-   **读取配置**: 在左侧边栏上传之前保存的 `.json` 文件即可瞬间恢复状态。

---

## 2. 基础桌面界面 (Tkinter) 🖥️

运行指令: `python -m src.ui.gui`

这是一个标准的桌面窗口版本。

### 使用说明
1.  **服务器与任务**: 在窗口顶部选择对应的服务器和任务 CSV 文件。
2.  **选择宠物**: 
    -   在“拥有宠物”列表中点击选择宠物（按住 Ctrl 或 Shift 可多选）。
    -   在“农场宠物”表格中，**双击**宠物的“数量”列，输入可借用的副本数。
3.  **开始计算**: 点击按钮，计算结果将显示在下方的文本框中。

---

## 查看结果

无论使用哪种界面，结果均包含：
-   **核心指标**: 显示总计层级奖励分和借用宠物总数。
-   **任务详情**: 每一个推荐的任务队伍及其原始得分、预计奖励等级（如：特阶、一阶、S 或 A）。

## 小贴士
-   计算器优先考虑跨越奖励阈值（如 37 分），而不是单纯堆砌原始分数。
-   在所有活跃任务中，总共最多只能借用 3 只宠物。
